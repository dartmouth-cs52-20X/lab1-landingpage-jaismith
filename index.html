How it works

Timetable Scanner CLI is built in Python3. This allows us to easily parse the Timetable as pure strings without DOM queries.

Twilio is a popular provider of communication APIs, including programmatic SMS. This allows us to text you as soon as we find an opening!

The Requests library is what allows us to access the Timetable website and retrieve its data. This is an extremely popular and well-maintained library.
